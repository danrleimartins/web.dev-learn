What is HTML?

Definition

HTML, short for Hypertext Markup Language is the name of that language responsible for giving the basic structure and the content of a webpage. Hypertext is text that contains hyperlinks, which in turn are those parts of a document which readers can click on to take them to a different document, using the link. You have all seen the blue underlined parts of a text, in not-so-good-looking web pages. This is a local hyperlink that takes you back to the home page. We can also have external hyperlinks, this one takes you to the Google homepage.

Brief History

HTML was created in 1991, by Tim Berners-Lee, at CERN (European Council for Nuclear Research) in Switzerland. Initially, HTML was designed to connect nearby research institutions and share documents with ease. In 1992, the development library WWW (World Wide Web), a worldwide network, was released, which together with HTML provided the worldwide use of the WEB.
Until 1995, CERN itself supervised and controlled the language standard. From that year on, the W3C (The World Wide Web Consortium), an entity that regulates Web standards, took over its supervision and control.

HTML Versions

Version 2.0 - 1995

The first version that emerged after the creation of pure HTML was 2.0. Its objective was to formalize all the HTML features that were already used. Its appearance took place at the first worldwide event on the Web, known as the World Wide Web Conference.

Version 3.2 - 1997

In addition to correcting compatibility problems in version 2.0, this version of the language provided the creation of tables, applets and floating text around images.

Version 4.01 - 1999

Its feature was to provide backward compatibility, but also to add support for multimedia options, style sheets and improve programming practices.

Version 5 - 2014 (current)

The latest version of HTML was developed to enhance the World Wide Web experience for developers and end users. HTML 5 provides high-level audio and video support that did not exist in previous versions.The main differences between HTML 5 and previous versions are:

  • HTML 5 uses a web database, while the other versions use the browser cache
  • HTML 5 is no longer based on SGML, so it can be improved and improved independently
  • This new version supports new types of form controls, such as date and time, e-mail, search, etc.

More About HTML History

Advantages

  • HTML is easy to write and use.
  • Allows the use of templates, which facilitates the design of web pages.
  • Loose syntax.
  • Widely used.
  • Supported on almost all browsers, if not all browsers.
  • HTML can integrate easily with other languages.
  • Free - you don't need to buy any software.
  • Very useful for beginners in the field of web design.

Disadvantages

  • It cannont produce dynamic output alone, as it's a static language.
  • The security features offered by HTML are limited.
  • You should keep track of obsolete tags and not use them.
  • It can only create static and simple pages.
  • Not centralized - each page has to be programmed separately.
  • Complexity - you need to write a lot of code to create a simple web page.

The <!DOCTYPE> Declaration

The <!DOCTYPE> declaration represents the document type, and helps browsers to display web pages correctly. It must only appear once, at the top of the page (before any HTML tags).
The declaration is not an HTML tag. It is an "information" to the browser about what document type to expect.The <!DOCTYPE> declaration is not case sensitive.
The <!DOCTYPE> declaration for HTML5 is: 

<!DOCTYPE html>


Fundamentals

Structure

An HTML file is rendered by the browser into a document with elements. Note the three elements in document: the <html> element, which encompasses the entire document; the <head> element, which is currently empty and provides information about the document; and the <body> element, which contains the part of the document that the web browser is intended to render.


Typical HTML Structure

Tags

A document in HTML consists of tags, with text in between them. There are opening and closing tags. Tags may not have a space between the opening angle bracket and the letters. It is customary when teaching a new language to begin with a simple program that simply outputs Hello, world! With HTML, it couldn’t be simpler. Open a new document in your editor and type in “Hello, world!”, as seen below: 

<h1> Hello, world! </h1>
